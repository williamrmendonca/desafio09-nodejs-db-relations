## :computer: Project

O Aplicativo é um pequeno e-commerce que permite a criação de clientes, produtos e pedidos, além de permitir ao cliente realizar novos pedidos, armazenando-os em um banco de dados postgres utilizando typeORM.

## :rocket: Built with

Este projeto foi desenvolvido com as seguintes tecnologias:

-   [Node.js](https://nodejs.org/)
-   [Express](https://expressjs.com/)
-   [TypeScript](https://github.com/microsoft/TypeScript)
-   [TypeORM](https://typeorm.io/)
-   [Tsyringe](https://github.com/microsoft/tsyringe)
-   [ESLint](https://eslint.org/)
-   [Prettier](https://prettier.io/)
-   [VS Code](https://code.visualstudio.com/)
